# GigShield AI – Parametric Income Insurance for Gig Workers

##  Problem Statement
India’s gig workers (Zomato, Swiggy, Zepto, etc.) lose 20–30% income due to disruptions like weather, pollution, and curfews. Currently, there is no insurance system to protect their income.

---

##  Our Solution
GigShield AI is an AI-powered parametric insurance platform that protects gig workers from income loss.

It automatically:
- Detects disruptions
- Triggers claims
- Provides instant payouts

---

##  Target Persona
Delivery Partners:
- Food delivery (Swiggy, Zomato)
- Grocery (Zepto, Blinkit)
- E-commerce (Amazon, Flipkart)

---

##  Key Features

###  AI Risk Assessment
- Predicts risk using weather & historical data
- Calculates **weekly premium dynamically**

###  Fraud Detection
- GPS validation
- Duplicate claim prevention
- Activity tracking

###  Parametric Insurance
- No manual claims
- Auto-trigger based on conditions:
  - Rainfall
  - Pollution (AQI)
  - Curfews

###  Instant Payout
- Automatic payout via payment gateway (mock)

---

##  Weekly Pricing Model

Premium is calculated based on:

- Location risk
- Weather probability
- Historical disruptions

Example:
- Low risk area → ₹20/week
- High risk area → ₹50/week

---

##  Workflow

1. User registers
2. AI calculates premium
3. User buys weekly plan
4. System monitors disruptions
5. Trigger event occurs
6. Claim auto-processed
7. Instant payout sent

---

##  AI Integration

- Dynamic pricing model
- Fraud detection using anomaly detection
